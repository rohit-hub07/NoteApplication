Notes Desktop App is a Next.js + Electron note manager that lets you create, search, and delete personal notes after signing in.

## What This App Does

- Lets users sign up, log in, and log out.
- Create notes with a title and description.
- View all notes in a grid with timestamps.
- Search notes by title or content.
- Delete notes with immediate UI updates.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- API routes for auth and notes

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.
